# Koyeb environment variables

## Current planned env vars
- `PORT` = `3000`
- `V3_PASSWORD_ENABLED` = `false` or `true`
- `V3_PASSWORD` = optional password for gate
- `V3_PASSWORD_HINT` = optional hint shown later in UI
- `V3_PROXY_ENABLED` = `false` or `true`

## Notes
- password gate is scaffolded at the backend level
- proxy status endpoint is scaffolded, but the real proxy runtime is not integrated yet
