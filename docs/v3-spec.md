# Skeezers Arcade V3 Spec

## Product goals
- Wide collection of apps and games
- Clean, easy-to-use UI
- Fast-feeling launcher shell
- Built-in tab system
- Optional password protection
- Various themes
- Inspect element
- Now.gg support
- GeForce NOW support
- Emulators
- Real proxy/backend support

## Core principles
1. Real hosted experiences over placeholders
2. One coherent platform shell
3. Modular integration boundaries
4. Backend-capable deployment from day one
5. Third-party repo ingestion must stay organized

## Lanes
- Games
- Apps
- Proxy
- Emulators
- Settings
- Tabs/Workspace

## MVP
- Koyeb-ready backend
- launcher shell
- apps lane
- games lane
- built-in tabs
- basic proxy flow
- settings/themes
- optional password gate

## Later phases
- inspect element
- now.gg/geforce support
- emulator lane
- admin/integration tooling
